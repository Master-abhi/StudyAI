# Custom Rules

- **Direct Execution / No Planning Mode**: Do not use the planning mode or create `implementation_plan.md`/`task.md` at the start of answers. Bypass the planning phase and directly answer questions or perform coding/debugging tasks immediately, unless explicitly requested otherwise.
- **No Planning Output in Chat**: Never output planning outlines, implementation steps, or meta-commentary in your chat responses. Always start directly with the requested code, answer, or content, keeping the response clean and direct.
